# 📈 AI Stock Analysis Dashboard

A modern AI-powered stock market analysis platform built using React, Material UI, Node.js, Express, and LLM APIs.

The application allows users to search stocks using natural language prompts and receive dynamically rendered AI-generated investment insights through a production-style finance dashboard interface.

---

# 🚀 Features

- AI-powered stock analysis engine
- Natural language stock search
- Dynamic frontend rendering
- Responsive dashboard architecture
- Analyst outlook section
- Risks & strengths visualization
- Financial performance insights
- Valuation metrics analysis
- Reusable React component architecture
- Modern dark-themed finance UI

---

# 🛠️ Tech Stack

## Frontend

- React.js

- JavaScript (ES6+)
- Axios / Fetch API

## Backend

- Node.js
- Express.js
- REST API Architecture

- Groq API / LLM APIs

# ⚙️ Installation

## Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

# Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```bash
http://localhost:5050
```

---

# 🔐 Environment Variables

Create `.env` inside backend folder:

```env
AI_KEY=your_api_key
AI_MODEL=llama-3.1-8b-instant
```
